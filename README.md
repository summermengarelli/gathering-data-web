# Gathering Data from the Web Using APIs

This is the workshop curriculum for "Gathering Data from the Web Using APIs." Assuming little to no prior knowledge about APIs, this lesson plan will introduce learners to some of the core features of APIs, including the etiquette of proper use, and will lead learners through hands-on examples that will give a taste for the possibilities of data collection via APIs.
